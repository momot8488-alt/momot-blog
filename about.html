关于 MoMot

思考，记录，分享。在喧嚣的互联网中保留一片属于自己的安静角落。

这个博客

MoMot 是一个个人博客，创建于 2026 年。这里没有算法推荐，没有广告干扰，只有纯粹的思考和记录。

博客采用纯静态方案构建——HTML + CSS + JavaScript，部署在 GitHub Pages 上，使用自定义域名 momot.top。技术栈极简，维护成本几乎为零，把精力集中在内容创作本身。

关于作者

我是一名热爱技术的创作者，关注前端开发、产品设计和数字工具。在工作之余，喜欢阅读、写作和探索互联网的边界。

我相信：

  • 好的工具让人专注于创造而非配置
  • 写作是最好的思考方式
  • 互联网的去中心化精神值得捍卫
  • 简洁是复杂的最终形态

技术栈

  • HTML5 语义化结构
  • CSS3 响应式设计（Grid + Flexbox）
  • 原生 JavaScript（零框架依赖）
